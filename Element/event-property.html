Element property:

大多数事件从文档层次结构冒泡到Document节点，然后从那儿再到Window对象。 Window对象有不少专有的事件处理程序，包括onblur, onerror, onfocus, onload, onscroll。 body元素的HTML属性注册的事件处理程序会注册在Window对象上。

这儿列出的大多数事件只会在特定类型的HTML元素上触发，不过由于这些事件大多会 在文档树中向上冒泡，因此这些事件处理程序属性一般是为所有元素定义的。 在HTML5的audio和video上出发的媒体事件不会冒泡。 一些与HTML5表单相关的事件也不会冒泡。
