Script private property:

  • boolean async
  • string charset
  • boolean defer
  • string src
  • string text
  • string type

note:

IE9+, chrome, firefox, opera 都支持async和defer

如果同时在一个script元素上使用两者，async属性会压过defer

script元素的解析执行流程

普通script元素：先解析script元素，然后停止解析，执行JS，然后继续解析

defer script元素： 在完成解析并关闭 </html> 节点之后，开始解析，然后执行

async script元素： 遇到该script后，另起线程下载文件，下载完后就解析执行；原线程继续解析；

在script元素的js中，不能操作还未解析的DOM节点，否则会报错。
